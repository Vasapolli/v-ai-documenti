# Modelli grafici dello Studio — DA FORNIRE

Questa cartella deve contenere tre file, che non sono nel repo perché sono materiale dello Studio:

| File | Cosa deve essere |
|---|---|
| `carta-intestata.docx` | Documento Word vuoto con intestazione, piè di pagina, font e stili dello Studio (Titolo, Titolo 2, Corpo, Citazione). Serve come base per pareri, circolari e memo. |
| `modello-slide.pptx` | Presentazione con il layout dello Studio: slide titolo, slide contenuto, slide fonti. |
| `modello-prospetto.xlsx` | Foglio con intestazioni formattate e i tre fogli previsti: Prospetto, Fonti, Lacune. |

**Finché mancano**, il plugin consegna in Markdown e lo dichiara: non deve inventare una grafica
dello Studio né imitarla a occhio.

Per un'altra installazione (altro studio), si sostituiscono questi tre file: è l'unica cosa da
cambiare oltre a `V_AI_MCP_URL`.
